# Eslint Config

<br>

## Requirements
- Node 10

<br>

## Module Installation

```sh
# using npm
$ npm install --save-dev @internet/eslint-config

# or using yarn
$ yarn add --dev @internet/eslint-config
```

<br>

## Usage

Add this to your eslint config:

```json
{
  "extends": ["@internet"]
}
```
